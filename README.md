# 📊 Customer Segmentation & Churn Prediction System

## 📌 About the Project
This project is a Machine Learning-based **Customer Segmentation and Churn Prediction System** designed to analyze customer data, understand distinct customer behavioral groups, and predict whether a customer is likely to leave a service.

The main objectives are to help businesses:
* **Identify distinct customer segments** using clustering techniques.
* **Predict customer churn behavior** using classification models.
* **Improve customer retention strategies** using data-driven insights.

This project covers the complete end-to-end Machine Learning workflow, including data preprocessing, exploratory data analysis (EDA), clustering, model building, and visualization.

---

## 🚀 Key Features
* **Data Cleaning & Preprocessing:** Handling missing values, managing outliers, and formatting data to prepare clean datasets.
* **Exploratory Data Analysis (EDA):** Understanding customer behavior patterns through statistical analysis and visualization.
* **Customer Segmentation (Clustering):** Using K-Means Clustering to divide customers into meaningful groups based on behavior.
* **Churn Prediction:** Building classification models to predict whether a customer will churn or not.
* **Feature Engineering:** Creating and selecting important features to improve model performance.
* **Data Visualization & Insights:** Visual representation of clusters and churn patterns using Matplotlib and Seaborn.
